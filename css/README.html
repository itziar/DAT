Resumen de las peculiaridades:

Creo que las peculiaridades que puede tener mi aplicacion son:

-Para desplegar los comentarios hay que hacer click en la flecha

-Los comentarios que llevan asociados una localizacion tienen puesto la palabra localizacion

-Los comentarios se resaltan usando otro color y en el mapa sólo se activa el último comentario abierto con un popup de la biblioteca leaflet

-Las pestañas se van activando cuando haya datos que mostrar mientras tanto están ocultas

-No se puede añadir dos veces a un mismo usuario, ni tampoco se puede enviar un formulario vacío

-Utilización de localstorage para guardar los ids de los usuarios

-Footer de copyright con el enlace al código en github (mostrado en la parte opcional)
